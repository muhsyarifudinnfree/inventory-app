# Inventory-App  
**Sistem Manajemen Aset & Pemeliharaan**  
Aplikasi berbasis web untuk pelacakan aset dan alur kerja pemeliharaan, dibangun dengan **React + Vite**, menggunakan Firebase, Cloudinary, dan Tailwind CSS.  
Tersedia tiga jenis peran pengguna: **Manager**, **Admin**, dan **Teknisi**.
